Twitter, Couchbase and amCharts

Pablo Casado (@hardlifeofapo)

How to start with node.js. An example for teaching purposes at UPC showing off basic node.js fundamentals with streaming twitter messages to Couchbase and visualise the data via socket.io in amCharts.

Hybrid web application architecture with Backbone, Rails and Node.js

Jordi Romero (@jordiromero)

Sometimes Rails is not enough, and Backbone can be used to delegate some of the logic to the client, consuming a REST API. In this talk Jordi will go through what it takes to progressively migrate from a server-side standard Rails app to a hybrid solution that leverages the best of both worlds.
